# document-page-template

Template for making a document page from a @yamanoku presentation.

## Install
```bash
npm ci
```

## Dev Server
```bash
npm run serve
```

### Japanese Page
```bash
http://localhost:8081/ja/
```

### English Page
```bash
http://localhost:8081/en/
```

## Build
```bash
npm run build
```

## Deploy
```bash
npm run deploy
```
